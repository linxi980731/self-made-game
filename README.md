# 杨柳青年画非遗传承文字游戏

## 游戏简介
基于中国非遗文化杨柳青木版年画创作的文字互动游戏，通过Ren'Py引擎开发。

## 运行方法
### 引擎驱动
下载并安装Ren'Py8.5.0
2.打开Ren'Py Launcher
选择“项目” -> “打开项目”
定位到本游戏文件夹
点击“启动项目”

### 直接运行
1.下载release中的对应版本
2.解压文件
3.运行“gengzinianhuahun.exe”

## 开发工具
**引擎**：Ren'Py 8.5.0
**编程语言**：Python + Ren'Py Script
**编辑器**：vscode
**版本控制**：Git + Github
